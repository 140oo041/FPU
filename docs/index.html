Tiny Tapeout Numeric Coprocessor

FPU-130 Design Specification

Build-ready specification for a compact bfloat16 floating-point unit targeting Tiny Tapeout. The design uses an iterative datapath, SPI command transport, optional burst-mode operand streaming, CRC-protected frames, and a 16-bit accumulator for chained operations.

Open Spec View Repository
Format bfloat16
Core Clock 40 MHz
SPI Mode 0, up to 10 MHz
Ops ADD SUB MUL DIV NEG ABS SLT NOP
